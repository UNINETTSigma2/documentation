# BUSCO
BUSCO is a tool to assess genome assembly and annotation completeness with
single-copy orthologs.

[To find out more, visit the BUSCO
website.](https://busco.ezlab.org/)

## Running BUSCO

| Module     | Version     |
| :------------- | :------------- |
| BUSCO | 3.0.2-intel-2018b-Python-2.7.15 <br> 4.0.5-foss-2019b-Python-3.7.4 <br> 4.1.4-foss-2020a-Python-3.8.2 <br> 5.0.0-foss-2020b |

To see available versions when logged into Saga issue command

    module spider BUSCO

To use BUSCO type

    module load BUSCO/<version>
