# AlphaFold
AlphaFold v2 is a protein structure prediction software based on a neural network
approach.

[To find out more, visit the AlphaFold
website.](https://github.com/deepmind/alphafold)

## Running AlphaFold

| Module     | Version     |
| :------------- | :------------- |
| AlphaFold | 2.0.0.1 <br>2.1.1|

To see available versions when logged into Saga, use:

    module spider alphafold

To use AlphaFold type

    module load Alphafold/<version>
